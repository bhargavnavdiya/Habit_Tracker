## Quick Start

### Option 1: Simple (No Backend - Uses LocalStorage)

Just open `index.html` in your web browser! All data is stored locally in your browser.

### Option 2: With Backend Server

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open your browser and navigate to:
```
http://localhost:3000
```

## Usage

1. **Monthly View**: Track your habits day by day
   - Click on any day to mark a habit as complete
   - Use the dropdown menus to track your mood and motivation
   - View your progress with interactive charts

2. **Yearly Statistics**: See your progress across all months
   - Navigate between years
   - View monthly breakdowns
   - Track overall trends

3. **Settings**: Manage your habits
   - Add new habits
   - Delete habits you no longer track
   - Export your data for backup
   - Import previously exported data
